## 📌 Project Overview

Car Rental MERN Web App is a full-stack car rental platform developed using the MERN stack architecture.  
The application allows users to browse available vehicles, view car details, securely authenticate, book cars, and manage rental activities through an interactive and responsive interface.

Users can also list their own cars for rent, manage their vehicle listings, and handle incoming booking requests from other users.

The frontend is built with React and Vite for fast performance and smooth user experience, while the backend is powered by Express.js and MongoDB for API handling, authentication, and database management.

The project is deployed using **Vercel** for seamless hosting and fast global delivery.

---

## 🚀 Features

- User authentication with JWT
- Password hashing using bcrypt
- Protected routes and secure access
- Browse and view available rental cars
- Book cars directly through the platform
- Manage personal bookings
- Add cars for rent
- Edit and manage listed cars
- View incoming booking requests for listed cars
- Approve or manage booking requests from other users
- RESTful API integration
- Image upload support
- Cloud image storage integration
- Responsive UI design
- Toast notifications
- Client-side routing
- Form validation
- Database integration with MongoDB
